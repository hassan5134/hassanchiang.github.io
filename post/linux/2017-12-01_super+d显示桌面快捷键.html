super + d 显示桌面快捷键

发表于 2017-12-01 更新于 2020-12-20 分类于 其他

如果某个 Linux 发行版没有super + d 显示桌面快捷键，可以自己配置一下

命令： wmctrl -k on

通常在 system - setting - keyboard 的 Custom 标签下进行设置
